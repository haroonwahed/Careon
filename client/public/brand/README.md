# Carelane agent-ready brand kit

## Recommended files
- Dark surfaces: `logos/png/carelane-logo-gradient-white-transparent.png`
- Light surfaces: `logos/png/carelane-logo-navy-transparent.png`
- Narrow navigation: `logos/png/carelane-logo-compact-transparent.png`
- Monochrome dark surfaces: `logos/png/carelane-logo-white-transparent.png`
- Standalone mark: `marks/carelane-mark-gradient-transparent.png`
- PWA icon: `icons/carelane-app-icon-512.png`
- Apple touch icon: `icons/carelane-app-icon-180.png`
- Browser favicon: `icons/favicon.ico`

## Installation
Copy the `carelane_brand_kit` folder to `public/brand`.

## Rules
1. Do not stretch, recolor or redraw the logo.
2. Use `object-fit: contain`.
3. Keep at least 12% of the logo height as clear space.
4. Prefer transparent assets in the application.
5. Use `*-bg` files only where the original full background artwork is required.
6. The `/source` folder contains untouched uploaded originals.

## SVG note
The supplied artwork is raster. No fake SVG wrappers are included. A true SVG requires the original vector master or a professional redraw.

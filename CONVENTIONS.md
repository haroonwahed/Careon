# Zorg OS Coding Conventions

Follow AGENTS.md as the source of truth.

## Core Behavior

Do not violate:
Casus -> Samenvatting -> Matching -> Aanbieder Beoordeling -> Plaatsing -> Intake

## Aider Working Rules

- Work on the smallest file set possible.
- Do not scan or rewrite the whole repo unless explicitly asked.
- Ask before touching unrelated files.
- Prefer root-cause fixes.
- Keep changes narrow but complete.
- Do not auto-commit.
- Show diffs clearly.
- Run or suggest the smallest meaningful test command.

## User-Facing Copy

Use Dutch terms:
- Casus
- Samenvatting
- Matching
- Aanbieder Beoordeling
- Plaatsing
- Intake
- Intake-overdracht
- Zorgaanbieder
- Gemeente
- Regiekamer
- Signalen
- Volgende beste actie

Avoid generic English copy in live UX.

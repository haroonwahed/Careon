# Regiekamer Skill

Use this skill for alerts, signals, escalation logic, intervention views, and SLA warnings.

Regiekamer is an intervention layer, not an analytics dashboard.

## Signal types should focus on:

- missing critical information
- absent summary
- no viable match
- weak match needing verification
- provider rejection
- overdue provider review
- capacity risk
- delayed placement
- delayed intake follow-up

## Every signal must answer:

- wat is er mis
- waarom is het belangrijk
- wie moet handelen
- welke actie is nu nodig
